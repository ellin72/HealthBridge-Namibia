# HealthBridge Namibia - User Guide

## Getting Started

### Registration
1. Navigate to the registration page
2. Fill in your details:
   - First Name
   - Last Name
   - Email Address
   - Phone Number (optional)
   - Password
   - Select your role (Patient, Healthcare Provider, Wellness Coach, or Student)
3. Click "Sign Up"
4. You'll be automatically logged in

### Login
1. Enter your email and password
2. Click "Sign In"
3. You'll be redirected to your dashboard

## User Roles

### Patient
Patients can:
- Book appointments with healthcare providers
- View consultation notes
- Access wellness content
- View learning resources

### Healthcare Provider
Healthcare providers can:
- View and manage appointments
- Create consultation notes
- Create assignments for students
- Grade student assignments

### Wellness Coach
Wellness coaches can:
- Create and manage wellness content
- Publish nutrition, fitness, and stress management articles
- Edit and delete their own content

### Student
Students can:
- Access learning resources
- View assignments
- Submit assignments
- View grades and feedback

## Features

### Telehealth Module

#### Booking an Appointment
1. Navigate to "Appointments"
2. Click "Book Appointment"
3. Select a healthcare provider
4. Choose date and time
5. Add any notes
6. Click "Book"

#### Managing Appointments
- View all your appointments on the Appointments page
- Providers can confirm or cancel appointments
- Patients can cancel their own appointments
- View appointment details and status

#### Consultation Notes
- Providers create notes after appointments
- Patients can view their consultation notes
- Notes include diagnosis, prescription, and follow-up information

### Wellness Hub

#### Viewing Content
1. Navigate to "Wellness Hub"
2. Browse content by category:
   - Nutrition
   - Fitness
   - Stress Management
3. Click on content to view details

#### Creating Content (Wellness Coaches)
1. Navigate to "Wellness Hub"
2. Click "Create Content"
3. Fill in:
   - Title
   - Description
   - Category
   - Content (full article)
   - Image URL (optional)
   - Video URL (optional)
4. Click "Create"

### Learning Zone

#### Accessing Resources
1. Navigate to "Learning Zone"
2. Browse available learning resources
3. Click "Download" to access PDFs and documents

#### Viewing Assignments (Students)
1. Navigate to "Learning Zone"
2. View all available assignments
3. See due dates and descriptions

#### Submitting Assignments (Students)
1. Find the assignment you want to submit
2. Click "Submit Assignment"
3. Upload your file (PDF, DOC, DOCX)
4. Click "Submit"

#### Creating Assignments (Instructors)
1. Navigate to "Learning Zone"
2. Click "Create Assignment"
3. Fill in:
   - Title
   - Description
   - Due Date
   - Upload assignment file (optional)
4. Click "Create"

#### Grading Assignments (Instructors)
1. View assignment submissions
2. Click on a submission
3. Enter grade and feedback
4. Click "Grade"

## Profile Management

### Updating Profile
1. Navigate to "Profile"
2. Update your information:
   - First Name
   - Last Name
   - Phone Number
3. Click "Update Profile"

### Changing Password
1. Go to Profile page
2. Enter new password in password field
3. Click "Update Profile"

The mobile app provides the same features as the web platform:

## Troubleshooting

### Can't Login
- Verify your email and password
- Check if your account is active
- Contact support if issues persist

### Appointment Not Showing
- Check appointment status
- Verify you're logged in with correct account
- Refresh the page

### File Upload Issues
- Ensure file is PDF, DOC, or DOCX
- Check file size (max 10MB)
- Try a different browser

### Content Not Loading
- Check your internet connection
- Clear browser cache
- Try refreshing the page
