# HealthBridge Frontend

React web application for HealthBridge Namibia.

## Setup

1. Install dependencies: `npm install`
2. Configure `.env` file with `VITE_API_URL`
3. Start dev server: `npm run dev`

## Features

- User authentication
- Appointment management
- Wellness Hub
- Learning Zone
- Profile management

## Tech Stack

- React 18
- TypeScript
- Material-UI
- Vite
- React Query

## Build

```bash
npm run build
```

Production build will be in `dist/` directory.
