# Contributing to HealthBridge Namibia

Thank you for your interest in contributing to HealthBridge Namibia!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Follow the setup guide in `docs/SETUP_GUIDE.md`
4. Create a new branch for your feature

## Code Style

- Use TypeScript for all new code
- Follow existing code patterns
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions small and focused

## Commit Messages

Use clear, descriptive commit messages:
- `feat: Add appointment reminder notifications`
- `fix: Resolve authentication token expiration issue`
- `docs: Update API documentation`
- `refactor: Simplify wellness content controller`

## Pull Request Process

1. Ensure all tests pass (when tests are added)
2. Update documentation if needed
3. Submit PR with clear description
4. Address review comments

## Project Structure

- `backend/`: API server code
- `frontend/`: Web application
- `mobile/`: Mobile application
- `docs/`: Documentation

## Questions?

Contact the development team or open an issue.
